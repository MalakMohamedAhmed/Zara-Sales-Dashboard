# 👗 Zara Sales Performance Dashboard
A real-time data visualization tool built with **Streamlit** to analyze sales trends, pricing strategies, and seasonal performance across global markets.

[[Link to Live App on Hugging Face Spaces](https://huggingface.co/spaces/MalakMohamed21/Zara_Dashboard)]

## 🚀 Features
* **Dynamic Filtering:** Filter data by Country of Origin, Product Section (Men/Women), and Price Range.
* **Distribution Analysis:** High-resolution KDE plots to understand price and revenue density.
* **KPI Metrics:** Instant view of Total Revenue, Average Price, and the highest-performing Season.
* **Smart Selection:** "Always All" logic—if no filter is selected, the dashboard automatically displays the full dataset to prevent empty charts.
* **Dark Mode Optimized:** Designed with a sleek, transparent UI for professional presentation.

## 🛠️ Tech Stack
* **Language:** Python
* **Framework:** [Streamlit](https://streamlit.io/)
* **Data Manipulation:** Pandas
* **Visualization:** Matplotlib & Seaborn
* **Deployment:** Hugging Face Spaces
